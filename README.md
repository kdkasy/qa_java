## Technologies 

- Java 11
- Maven 3.9.6

## Dependencies 

- Jacoco 0.8.7
- Mockito 3.12.4
- Junit 4.13.2

## Installing

1. Download project 
2. Compile with maven <br> ```$ mvn compile ```
3. Run tests <br> ```$ mvn clean test```
4. To generate coverage report use <br> ```$ mvn verify```
